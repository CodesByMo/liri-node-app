# How to use
Include search parameters after each of the following commands.
* `concert-this + band name`
+ `spotifiy-this-song + song name`
- `movie-this + movie name`
* `do-what-it-says`


# Technologies used
* Javascript
+ Node.js
- OMDB API
* BandsInTown API
+ Spotify API
